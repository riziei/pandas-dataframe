# 🧠 Machine Learning Practice Projects (scikit-learn)

This repository contains beginner-friendly Machine Learning classification projects using built-in datasets from `scikit-learn`.

---

# Projects Included

## Iris Flower Classification
**Goal:** Predict the species of an iris flower based on its measurements.


- Dataset: Iris dataset
- Problem: Predict flower species

### Classes:
- Setosa
- Versicolor
- Virginica

### Features:
- Sepal length
- Sepal width
- Petal length
- Petal width

---

## Wine Classification
**Goal:** Classify wine into different categories based on chemical properties.


- Dataset: Wine dataset
- Problem: Classify wine type based on chemical properties

### Classes:
- Class 0
- Class 1
- Class 2

### Features:
- Alcohol
- Malic acid
- Ash
- Alcalinity of ash
- Magnesium
- Flavanoids
- Color intensity
- Proline
- And more

## 🫀 Breast Cancer Classification
**Goal:** Predict whether a tumor is malignant or benign.


- Dataset: Breast cancer dataset
- Problem: Predict tumor type

### Classes:
- Malignant (0)
- Benign (1)

### Features:
- Mean radius
- Mean texture
- Mean smoothness
- Mean area
- Mean compactness
- And more medical features
